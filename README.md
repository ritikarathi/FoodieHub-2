# 🍔 FoodieHub

FoodieHub is a modern full-stack food ordering web application that allows users to explore delicious food items, browse categories, add items to their cart, and place orders through an intuitive and responsive interface.

## ✨ Features

- 🍕 Browse food items by category
- 🔍 Explore curated food collections
- 🛒 Add and remove items from the shopping cart
- 👤 User authentication (Login & Register)
- 📱 Fully responsive user interface
- ⚡ Fast and interactive React frontend
- 🗂️ RESTful backend APIs using Spring Boot
- 🗃️ Database integration for food, users, and orders
- 📦 Admin panel for managing food items (Upcoming)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/FoodieHub.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
mvn spring-boot:run
```
